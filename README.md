# TicTacToe
An interactive one or two player game

## Usage
1. Enter '**S**' to begin the game.
2. Enter '**1**' for one player mode or '**2**' for two player mode.
3. Play the game, entering a move betwween '**A~C**' and '**1-3**'.
4. Enter '**Q**' to quit the game
